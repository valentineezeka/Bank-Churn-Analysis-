# 🏦 Bank Customer Churn Analysis

## 📌 Project Overview
A team data analytics project exploring why bank customers leave using a real-world dataset of 10,000+ customers.

## 🎯 Objectives
- Clean and prepare raw banking data for analysis
- Identify key factors driving customer churn
- Visualize patterns across geography, age, and account activity

## 🛠️ Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib)
- **Google Colab**
- **Excel**

## 📊 Key Findings
- Germany had the highest churn rate among the three countries
- Customers aged 45–55 were most likely to exit
- ~20% overall churn rate in the dataset
